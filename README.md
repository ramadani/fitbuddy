# fitbuddy

Golang sport running package

### Installation

```
go get github.com/ramadani/fitbuddy
```

## Functions

- [x] Get max heart rate
- [x] Get heart rate zone
- [x] Pace Calculator
- [ ] Pace Converter
- [ ] Negative Split Calculator
- [ ] Positive Split Calculator
